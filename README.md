# Portfolio Website

A modern portfolio website built with React, TypeScript, and Tailwind CSS.

## Quick Deploy

### Option 1: Use the automated script
Double-click `QUICK-DEPLOY.bat` and follow the prompts.

### Option 2: Manual deployment

#### 1. Push to GitHub
```bash
git push -u origin main
```

#### 2. Deploy to Vercel
1. Go to [vercel.com/new](https://vercel.com/new)
2. Import your `sitewebportofolio` repository
3. Vercel will auto-detect Vite settings
4. Click "Deploy"

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

- React 18

- Tailwind CSS
